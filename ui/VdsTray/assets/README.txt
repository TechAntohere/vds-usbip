Drop controller art here (PNG, transparent background):

  dualsense.png        - regular white DualSense (front view)
  dualsense_edge.png   - DualSense Edge (front view)

The tray app loads these at runtime from the folder next to the exe, so you
can replace them with higher-resolution renders anytime without rebuilding.
If a file is missing, the popup just shows without the controller image.
